# Carpe Diem

A Next.js todo app with Supabase, PostgreSQL, and GitHub authentication.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- **Next.js** - React framework
- **Supabase** - PostgreSQL database & auth
- **Tailwind CSS** - Styling
- **Vercel** - Deployment

## Deploy on Vercel

Deploy to [Vercel](https://vercel.com) by connecting your GitHub repository. No root directory configuration needed.
